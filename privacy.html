Politique de confidentialité

Dernière mise à jour : 6 décembre 2024

1. Absence de collecte de données

Cette application ne collecte, ne stocke et ne partage aucune donnée personnelle ou autre information des utilisateurs.

2. Utilisation des API tierces

L'application utilise des API tierces pour fournir ses fonctionnalités. Toutes les clés API utilisées sont privées et stockées localement sur le dispositif de l'utilisateur (par exemple, un Raspberry Pi). Les interactions avec ces API sont sécurisées.

3. Sécurité

Bien que l'application soit conçue pour fonctionner en local, aucune garantie totale de sécurité ne peut être donnée en cas d'utilisation malveillante ou de modifications non autorisées.

4. Modifications

Nous nous réservons le droit de modifier cette politique de confidentialité. Veuillez vérifier cette page régulièrement pour prendre connaissance des mises à jour.
